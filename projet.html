Chapitre 4 : Internt et Web 

	

	1- Introduction:

		Le web et internet c’est pareil :
		Le WWW c'est pour WORLD WIDE WEB
		sur Internet. Le Web permet de consulter avec un navigateur, des pages accessibles sur des
		sites.
		Le Web a été inventé par Tim Berners-Lee et
		Robert Cailliau plusieurs années après Internet
		Internet : c'est l'infrastructure et les devices et les services(Mail,www,...)
		Web : resources circulant sur l’Internet (video,image,page,application...).

	1-1- Principe World Wide Web :

		Accéder à des documents textuels,Situés sur des machines accessibles par Internet (serveur public)
		Reliés entre eux par les lien « hypertexte » (ou bien hyperlinks)
		Actuellement : servir des ressources
		De différentes type : texte, image, son, vidéo, contenu applicatif...et interactif
		(l’achat/discussion....)

	1-2-URL (pour Uniform Resource Locator)

		 Un lien hypertexte est un lien qui permet de suivre une URL (ouvrir une nouvelle page).
		 URL (pour Uniform Resource Locator), est le nom unique de la page. décrivant l’emplacement
		 d’une ressource. elle contient :
		 L’adresse du serveur (domaine ou bien address IP) et l’adresse de la page sur le serveur (URI)
		 Exemple: http://example.com/un/chemin/page.html
				  http://127.0.1.1/var/www/html/index.htùl

	2- Aspects techniques du Web

		Les 3 mécanismes de base du Web sont :
		2-1- URL :
			 Le mécanisme de localisation des resources peut faire appel au protocole DNS
		2-2- HTTP : Protocole de niveau applicatif qui dirige communication entre client et serveur
			(request ,response) utilisons des mthod (GET POST PUT...)
		2-3- HTML :
			 Langage de description et structure  de  pages Web 

	3- Page web et site web :

		La page-web c'est un document(image,txt,lien,video) contient des éléments structuré par html format
		stocké dans un serveur dans un fichier : html (ou .htm, .html5) présntée (render) par le navigateur
		apres l'interpretation ,site web c'est la collection des pages web reliées entre elles par des liens 
		hypertextes avec un chemin commun ou relatif (comme le disque local voir 1-2) 

	4) Création d’une page web HTML :

		HTML : HyperText Markup Language Langage 
		Un document HTML est :
		Un fichier texte, construit suivant une syntaxe précise avec l'extension .html .html5 .htm ou htmlx
		Structuré par des balises (tag) <balise></balise>
		Interprété (visualisé) dans un navigateur (Mozilla Firefox, Internet Explorer, Safari,
		etc.)
		Créé dans un éditeur de pages web ou tout éditeur de texte
		(Sublime Text,notepad)

	 Les principes de langage HTML:

		Les balises (<html><meta>...)
		Les principes de langage HTML
		<html>: Délimite le début et la fin du document
		<head>Entête du document
		<title> ... </title>: Titre du document 
		</head>
		<body> contenu de page </body> : Corps du document
		Un fichier HTML doit comporter les balises précédente au minimum
		Les titres et paragraphes
		<hn> ... <hn> : 
		Titre de niveau n :1 à 6. Exemple <h1> Grand titre </h1&&h6>plus petit</h6>
		<p> ... <p> : paragraphe

	exemple : tapez ctrl+U pour voir un exemple réelle
